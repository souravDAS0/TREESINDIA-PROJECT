TREESINDIA is a **unified digital platform** with three main modules: **Home Services**, **Construction Services**, and **Marketplace**. The platform features simplified phone+OTP authentication, advanced booking systems with time slot management, real-time worker tracking, and subscription-based marketplace features.

## üîê **How Does It Work?**

### **Simplified Authentication Process:**

```
1. User enters phone number
2. System sends OTP via SMS
3. User enters OTP
4. System validates and creates/logs in user
5. User accesses three main modules
```

### **Home Services Booking Process:**

#### **Fixed Price Services:**

```
1. User selects service category ‚Üí View available services
2. Choose service ‚Üí View fixed pricing and time slots
3. Select time slot ‚Üí Check worker availability
4. Book service ‚Üí Payment through Razorpay
5. Admin assigns worker ‚Üí User gets notification
6. Worker arrives ‚Üí Live location tracking
7. Service completion ‚Üí OTP verification and review
```

#### **Inquiry-based Services:**

```
1. User submits inquiry ‚Üí Chat with admin
2. Discuss requirements ‚Üí Admin understands needs
3. Admin sends quote ‚Üí Detailed cost breakdown
4. User accepts quote ‚Üí Book with available time slot
5. Payment processing ‚Üí Razorpay integration
6. Service execution ‚Üí Worker assignment and tracking
7. Service completion ‚Üí Verification and feedback
```

### **Construction Services Process:**

```
1. User books consultation ‚Üí Admin receives request
2. Requirement discussion ‚Üí Chat about project details
3. Admin sends quote ‚Üí Custom pricing and timeline
4. User accepts quote ‚Üí Payment and booking confirmation
5. Project execution ‚Üí Regular updates and communication
6. Project completion ‚Üí Final verification and payment
```

### **Marketplace Process:**

#### **Free Users:**

```
1. Browse properties ‚Üí Search and filter options
2. Contact owners ‚Üí Direct communication
3. Property viewing ‚Üí Schedule visits
4. Transaction ‚Üí Direct payment and documentation
```

#### **Subscribed Users:**

```
1. Access enhanced features ‚Üí Vendor lists and workforce
2. Direct communication ‚Üí Chat with vendors and workers
3. Project management ‚Üí List projects and requirements
4. Service booking ‚Üí Book services directly
5. Payment processing ‚Üí Secure transactions
```
